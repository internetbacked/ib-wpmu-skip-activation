Allow admin add user in Wordpress Multisite and skip the activation.

== Description ==

So often user unable to activate account. This issue remain even you installed this plugin but if your admin wants to help user or customer to add user manually, yes! This is the plugin come into the place.

A few notes about the sections above:

*   Main Site > Users > Add New > Add New User (not Add Existing User)

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
